# HTML Email Signature

A clean, professional HTML email signature with simple typography and neutral colors.

## Features
- Black & gray color scheme  
- Email-client safe (Gmail, Outlook)  
- Name, role, contact info, and social links  
- Lightweight table-based layout  

## Usage
Copy the HTML file and paste it into your email client’s signature editor.
